**Quiz Бот в Telegram (Название: BiBopQuiz)**\
Новый пользователь нажимает старт и у него появляется меню с тремя кнопками.\
Кнопка "Начать игру" запускает quiz и пользователь отвечает на вопросы.\
Если пользователь ответил верно, то ему приходит сообщение в котором написаны: вариант, выбранный пользователем.\
Если польщователь ответил неверно, то ему проходит сообщение о том, что он ошибся, вариант, который он выбрал и правильный ответ.\
Так же при каждом завершении квиза, польщователь может увидеть свою статистику.\
На нажатие кнопки "Моя статистика" можно увидеть данные о своей последней попытке.\
На нажатие кнопки "Таблица лидеров" пользователю показывается 10 других человек, с сортировкой по убыванию рейтинга.
